📊 Championship Dominance Ranking

SELECT team_name, year, wins, losses,
  ROUND(win_pct * 100, 1) as win_pct,
  RANK() OVER (ORDER BY win_pct DESC) as dominance_rank
FROM championship_teams
ORDER BY win_pct DESC

📋 Methodology

Data Sources: Team statistics scraped from Basketball-Reference.com for NBA champions from 2020-2024.

Database: SQLite database with normalized schema (championship_teams table) containing 14 key performance metrics per team.

SQL Techniques Used:

  • Window functions (RANK() OVER) for comparative rankings
  • CASE statements for categorical classification
  • Aggregate functions and calculated metrics (win percentage, ratios)
  • Multi-criteria composite scoring formula

Key Finding: The 2024 Celtics rank #1 across multiple dimensions: highest win percentage (78.0%), most points per game (120.6), and best composite championship score (88.6), establishing them as the most dominant recent NBA champion.
